## 学术论文

### 基准测试

- **SWE-bench: Can Language Models Resolve Real-world GitHub Issues?** - Jimenez et al., 2024. 定义了 coding agent 评测的标准基准，包含 GitHub 真实 issue 的自动修复任务。[论文链接](https://arxiv.org/abs/2310.06770)
- **SWE-bench Verified** - 经人工验证的 SWE-bench 子集，提供更可靠的评测结果。[网站](https://www.swebench.com/)

### Agent 架构

- **SWE-agent: Agent-Computer Interfaces Enable Automated Software Engineering** - Yang et al. (Princeton), 2024. 提出 Agent-Computer Interface (ACI) 概念，设计了针对软件工程任务优化的 agent 交互方式。[论文链接](https://arxiv.org/abs/2405.15793)
- **ReAct: Synergizing Reasoning and Acting in Language Models** - Yao et al., 2023. 提出 ReAct 框架，将推理（思维链）与行动（工具调用）交替进行，是现代 coding agent 的核心架构基础。[论文链接](https://arxiv.org/abs/2210.03629)
- **OpenDevin: An Open Platform for AI Software Developers** - Wang et al., 2024. OpenHands 的学术论文，描述了开放的 AI 软件开发平台架构。[论文链接](https://arxiv.org/abs/2407.16741)

### 工具使用

- **Toolformer: Language Models Can Teach Themselves to Use Tools** - Schick et al., 2023. 探索让语言模型自主学习使用外部工具的方法。[论文链接](https://arxiv.org/abs/2302.04761)
- **Tool Learning with Large Language Models: A Survey** - Qin et al., 2024. 大语言模型工具使用的综述论文，涵盖工具检索、调用与组合。[论文链接](https://arxiv.org/abs/2405.17935)
- **MCTS-Tool: Boosting LLM Tool Planning with Monte Carlo Tree Search** - 使用蒙特卡洛树搜索优化 agent 工具规划能力。

### 代码生成与理解

- **Evaluating Large Language Models Trained on Code** - Chen et al. (OpenAI), 2021. Codex 原始论文，奠定了 AI 代码生成的基础。[论文链接](https://arxiv.org/abs/2107.03374)
- **CodeAgent: Autonomous Communicative Agents for Code Review** - 代码审查自动化 agent 的研究。
- **RepairAgent: An Autonomous, LLM-Based Agent for Program Repair** - 自动程序修复 agent 的研究。

## 相关工具生态

### MCP 服务器集合

- [官方 MCP Servers](https://github.com/modelcontextprotocol/servers) - Model Context Protocol 官方维护的服务器集合，包含 filesystem、GitHub、PostgreSQL 等
- [awesome-mcp-servers](https://github.com/punkpeye/awesome-mcp-servers) - 社区维护的 MCP 服务器列表
- [MCP Hub](https://mcphub.io/) - MCP 服务器发现与安装平台
- [Smithery](https://smithery.ai/) - MCP 服务器注册与管理平台
- 常用 MCP 服务器：
  - `@modelcontextprotocol/server-filesystem` - 文件系统操作
  - `@modelcontextprotocol/server-github` - GitHub API 集成
  - `@modelcontextprotocol/server-postgres` - PostgreSQL 数据库访问
  - `@modelcontextprotocol/server-brave-search` - Brave 搜索集成
  - `@modelcontextprotocol/server-puppeteer` - 浏览器自动化

### LSP 服务器列表

- [Language Server Protocol 官方列表](https://microsoft.github.io/language-server-protocol/implementors/servers/) - 各编程语言的 LSP 服务器实现
- 常用 LSP 服务器（coding agent 常依赖这些进行代码分析）：
  - `typescript-language-server` - TypeScript/JavaScript
  - `pyright` / `pylsp` - Python
  - `rust-analyzer` - Rust
  - `gopls` - Go
  - `clangd` - C/C++
  - `jdtls` - Java

### Agent 框架

- [LangChain](https://github.com/langchain-ai/langchain) - LLM 应用开发框架
- [LangGraph](https://github.com/langchain-ai/langgraph) - 有状态 agent 编排
- [AutoGen](https://github.com/microsoft/autogen) - 多 agent 对话框架（Microsoft）
- [CrewAI](https://github.com/crewAIInc/crewAI) - 多 agent 协作框架
